# 🚀 پنل ایلیا
**فوق‌حرفه‌ای‌ترین پنل مدیریت Xray با ۷۰+ قابلیت**

## ✨ قابلیت‌ها
- اسکنر هوشمند CDN (Cloudflare/Fastly/Gcore)
- پشتیبانی از Hysteria2، TUIC، gRPC، XHTTP
- VLESS+Reality پسا-کوانتومی
- Auto-Pilot، تشخیص اختلال، Fragment Finder
- رابط کاربری فوق‌حرفه‌ای با افکت‌های Glassmorphism
- PostgreSQL، بکاپ خودکار، ربات تلگرام

## 🚀 استقرار روی Railway
1. مخزن را به گیت‌هاب آپلود کنید
2. در Railway: New Project → Deploy from GitHub repo
3. آدرس: `https://your-app.up.railway.app:2053`
4. ورود: `admin` / `admin`
